Installation Guide

⚠️ Important Warnings

  • You may need to enable "Install from Unknown Sources" in your device settings
  • Your device might display a warning about installing apps from outside the Play Store
  • Some antivirus apps might flag the APK download - this is a false positive

Step-by-Step Installation

  1. 1. Enable Unknown Sources:
    Go to Settings → Security → Enable "Unknown Sources" or "Install Unknown Apps"
  2. 2. Download the APK:
    Click the "Download App" button above and wait for the download to complete
  3. 3. Open the APK:
    Tap on the downloaded file in your notifications or file manager
  4. 4. Install:
    Tap "Install" when prompted. You may see a warning - this is normal for apps from outside the Play Store
  5. 5. Open the App:
    Once installed, tap "Open" to start using the Titan Institute PSC Quiz App

Troubleshooting

If you encounter any issues during installation:

  • Make sure you have enough storage space
  • Check if you have enabled installation from unknown sources
  • Try downloading the APK again
  • If problems persist, contact our support team
